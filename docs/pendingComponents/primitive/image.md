# UImage

## Properties

### src

* type: String

### alt

* type: String

### defaultSrc

* type: String

### visible
* type: boolean

### ratio
* type: number
Calculated as width/height, so for a 1920x1080px image this will be 1.7778. Will be calculated automatically if omitted

### min-width
* type: number

### min-hight
* type: number

## Examples

```html
<u-image src="api/getimage" ratio="1.77"  visible="true" @click="" />
```
