# Ulink(must confirm!!!!)

## Properties

### href

* type: String

### target 

* type: String
* available values: _self,_blank,_parent,_top

### title

* type: String

### id

* type: String

### text

* type: String

### cssClass

* type: String


### disable

* type: boolean
* default: false

### dir

* type: string
* default: left
* available values: left, right

### visible

* type: boolean
* default: true


### language

* type: String
* default: null
* available values: en-US, fa-IR, ...

## Methods

onclick


## Examples

```html
<u-link href="https://www.google.com" title="Click Me" target="_blank" text="Google" @click="" />
```
