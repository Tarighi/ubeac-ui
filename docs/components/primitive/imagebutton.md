# UImagebutton

## Properties

### src

* type: String

### alt

* type: String

### tooltip

* type: String

### id

* type: String

### text

* type: String

### cssClass

* type: String

### isreload

* type: boolean
* default: false

### disable

* type: boolean
* default: false

### dir

* type: string
* default: left
* available values: left, right

### visible

* type: boolean
* default: true


### language

* type: String
* default: null
* available values: en-US, fa-IR, ...

## Methods

click


## Examples

```html
<u-imagebutton src="~/image.png" id="btnSubmit" dir="right" isreload="true" tooltip="Click Me" @click="" />
```
